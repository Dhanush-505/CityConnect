## Key Features & How-To Instructions

### 1. View Assigned Tasks
1. Navigate to `/field-worker/tasks`.
2. Review assigned grievances, location addresses, priority flags, and expected completion dates.

### 2. Update Progress & Work Status
1. Click **Update Status**.
2. Select status (*Accepted → Travelling → Work Started → In Progress → Completed*).
3. Add progress remarks and upload progress photos.

### 3. Complete Task & Upload Evidence
1. Once repairs are completed, select **Completed**.
2. Upload clear completion evidence photos.
3. Submit update for administrative verification.
